Основные определения и понятия

Для начала давайте введем необходимые базовые термины, которые пригодятся нам в дальнейшем.

  • Конечное множество объектов называется словарем.

  • Элементы словаря называются символами.

  • Цепочкой (символов) над словарем будем называть произвольную конечную последовательность символов словаря. Символы будем далее обозначать латинскими буквами начала словаря, цепочки – малыми греческими буквами.
    Пример. Пусть Σ – словарь, содержащий три символа, a,b,c. Над словарем Σ можно построить различные цепочки, например, α=abbca, β=cca, γ=b.

  • Пустая цепочка, для ее обозначения введен специальный символ ε.

Операции над цепочками

  • Цепочки строятся из символов конечного словаря.

  • Пусть Σ – словарь, содержащий три символа: a, b, c. Σ={a, b, c}.
    Над словарем Σ можно построить различные цепочки, например, α=abbca, β=cca, γ=b.

  • Операция конкатенации двух цепочек: αβ=abbcacca.

  • Пустая цепочка – единица конкатенации: εα= αε = α.

  • Операция подстановки в цепочку вместо одной подцепочки другой цепочки.
    Например, цепочку abbcacca по правилу замены сас → b можно преобразовать в цепочку abbbca. Это можно записать: abbcacca ⇒ abbbca.

Язык: формальное определение

Определение.
Языком над словарем Σ называется произвольное множество цепочек над этим словарем.
Это определение достаточно широко. Оно задает язык как некоторое множество (обычно бесконечное) одномерных последовательностей элементов любой природы и вложенного в них смысла, причем элементы эти берутся из некоторого конечного набора (словаря).

Очевидно, что над конечным непустым словарем можно определить бесконечное количество цепочек конечной длины (счетное множество).

Очевидно, что над конечным непустым словарем можно определить бесконечное количество языков, т.е. подмножеств множества всех возможных цепочек (континуум, т.к. число подмножеств счетного множества имеет мощность континуум).

Примеры языков.
Регулярные множества. Формальное определение

  • Определение.
    Класс регулярных множеств над конечным словарем Σ определяется так:

    ∅ - регулярное множество;
    {ε} - регулярное множество;
    {a} - регулярное множество для любого a∈Σ;

  • Если S и T - регулярные множества, то регулярны и:

    объединение S∪T;
    произведение ST;
    итерация S* ;

  • Примеры регулярных множеств:

    {ab, ba}* {aa} = {aa, abaa, baaa, ababbaaa, babaabaa, ... }
    {b} ( {c} ∪ {d, ab}*) = {bc, b, bd, bab, bababd, ... }

  • Примеры нерегулярных множеств:

    {anbn| n>0} = {ab, aabb, aaabbb, aaaabbbb, ... };
    {α⏐ в цепочке α количества вхождений символов а и b совпадают} =
    = {aababbba, ε, baababaabb, ba, ab, ... }

Регулярные выражения

  • Регулярные множества хороши тем, что их можно описать формулами, которые называются регулярными выражениями.

  • Регулярное выражение показывает, как можно построить данное регулярное множество с использованием трех операций: конкатенации, объединения и итерации

  • Определение.
    Класс регулярных выражений над конечным словарем Σ определяется так:

    ∅, ε - и любое a∈Σ - регулярные выражения;

  • Если R1 и R2- рег. выр., то регулярными выражениями будут:

    их сумма R1 + R2; (знак ‘+’, иногда записывают как ‘|’)
    Например, {"ab", "c"}|{"ab", "d", "ef"} = {"ab", "c", "d", "ef"}

    их произведение R1R2;
    Например, {"boy", "girl"}{"friend", "cott"} = {"boyfriend", "girlfriend", "boycott", "girlcott"}

    итерация R1* и урезанная итерация R1+;
    Например, {"Run", "Forrest"}* = {ε, "Run", "Forrest", "RunRun", "RunForrest", "ForrestRun", "ForrestForrest", ...}

  • Приоритеты операций: *, произведение, сложение

  • Скобки ( ) используются для группировки (для задания порядка выполнения операций)

Пример регулярных выражений: ab + ba*; (aa)*b + (c + dab)*

Регулярные множества и регулярные выражения

Чтобы лучше понять операции, рассмотрим соответствие операций регулярных выражений и регулярных множеств:

  • конкатенация подформул соответствует произведению множеств

  • операция сложения ‘+’ (или) соответствует операции ∪ объединения множеств

  • операция итерации R* соответствует операции MR* итерации множеств (т.е. 0 или большее число повторений)

  • операция R+ соответствует операции MR+ усеченной итерации множества (т.е. 1 или большее число повторений)

Таким образом, выражение b( c + (d + a)*) определяет язык {bc, b, bd, bdd, bda, baad, ... }, полученный последовательностью операций над одноэлементными множествами: {b}•( {c} ∪ ({d} ∪ {a}•{b})*)
