Эквивалентность регулярных выражений

Два регулярных выражения \(R\) и \(S\) называются эквивалентными (обозначается \(R=S\) тогда и только тогда, когда соответствующие им языки совпадают, т.е. \(L(R)=L(S)\).

Пример:
\( \large a^*a^+ = aa^* = a^+\) - все непустые цепочки, состоящие из \( \large a\)

Для любых регулярных выражений \(R, S\) и \(T\) справедливы соотношения:

  • \(R + S = S + R\)

  • \(R + R=R\)

  • \(\emptyset + R = R\)

  • \((R + S) + T = R + (S + T)\)

  • \(R \varepsilon = \varepsilon R = R\)

  • \((RS)T=R(ST);\)

  • \(∅R=R∅=∅ (\)но в общем случае: \(RS ≠ SR);\)

  • \(R(S+T)=RS+RT;\)

  • \((S+T)R=SR+TR;\)

  • \(R^* = \varepsilon + R + R^2 + R^3 + .. + R^kR^*;\)

  • \(RR^*=R^*R;\)

  • \(R(SR)^*=(RS)^*R\)

  • \(R^+ = R + R^2 + R^3 + ... + R^kR^*\)

  • \(R^+ + ε = R^*\)

Пример эквивалентных преобразований:

\(b(b + aa^*b)\) => [ по правилу \(R = ε R\) ]

\(b(εb + aa^*b)\) => [ по правилу \(SR + TR = (S + T) R\) ]

\(b( ε + aa^* )b\) => [ по правилу \(RR^* = R^+\) ]

\(b( ε + a^+ )b\) => [ по правилу \(ε + R^+ = R^*\) ]

\(b(a^*)b\) => [ убираем скобки ]

\(ba^*b\)

Регулярные языки и автоматные языки

Итак, регулярные выражения - это конечные формулы, задающие регулярные языки. Но подобным же свойством обладают и конечные автоматы - они тоже задают языки

Теорема Клини:

Классы регулярных множеств и автоматных языков совпадают, т.е. \(L_A = L_R\).

Это значит, что любой язык, распознаваемый конечным автоматом, может быть задан формулой, и для любого языка, заданного формулой (рег. выр), может быть построен КА, его распознающий

*Источник: "Логика и теория автоматов" (Ю.Г.Карпов)
